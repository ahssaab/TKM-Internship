**SR Flip-Flop – Overview**

An SR (Set-Reset) Flip-Flop is a basic sequential logic circuit used to store one bit of data. It has two inputs, Set (S) and Reset (R), which control the output state by setting it to 1 or resetting it to 0. SR flip-flops are commonly used in memory and control circuits.
